# 🏡 Estate Agents Management System

A modern web-based system designed to help estate agents manage properties, tenants, complaints, and communication efficiently.

---

## 📌 Features

* 🏠 Property listing and management
* 👥 Tenant management system
* 📝 Complaint submission and tracking
* 💬 Real-time chat between tenants and management
* 🔔 Notification system for updates and alerts
* 🔐 Secure authentication (login & registration)
* 📊 Clean and user-friendly dashboard

---

## 🚀 Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP / Localhost

## ⚙️ Installation Guide

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/estate-management-system.git
   ```

2. Move the project to your server directory:

   * For XAMPP: `htdocs/`

3. Import the database:

   * Open phpMyAdmin
   * Create a database
   * Import `schema.sql`

4. Configure database connection:

   * Open your config file
   * Update the database credentials:

     ```php
     DB_HOST=localhost
     DB_USER=root
     DB_PASS=
     DB_NAME=your_database_name
     ```

5. Run the system:

   ```
   http://localhost/estate-management-system
   ```
