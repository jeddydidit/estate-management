# 🏡 Estate Agents Management System

A modern web-based system designed to help estate agents manage properties, tenants, complaints, and communication efficiently.

---

## 📌 Features

* 🏠 Property listing and management
* 👥 Tenant management system
* 📝 Complaint submission and tracking
* 💬 Real-time chat between tenants and management
* 🔔 Notification system for updates and alerts
* 🔐 Secure authentication (login & registration)
* 📊 Clean and user-friendly dashboard

---

## 🚀 Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP / Localhost


## 🎯 Purpose

This system was developed to improve the management of estates by:

* Reducing manual record keeping
* Improving communication between tenants and management
* Providing real-time updates and tracking
